Seletores

Seletores são usados para selecionar os elementos HTML que queremos estilizar ou manipular com JavaScript. Existem vários tipos de seletores, incluindo:

  • Seletores de tipo: selecionam elementos pelo nome da tag. Exemplo: p para selecionar todos os parágrafos. 
    p {
    color: blue;
}

  • Seletores de classe: selecionam elementos com uma classe específica. Exemplo: .minha-classe para selecionar todos os elementos com a classe "minha-classe". 
    .minha-classe {
    font-size: 18px;
}

  • Seletores de ID: selecionam um elemento com um ID específico. Exemplo: #meu-id para selecionar o elemento com o ID "meu-id".
    #meu-id {
    background-color: yellow; 
}

  • Seletores de atributo: selecionam elementos com um atributo específico. Exemplo: [type="text"] para selecionar todos os elementos de entrada do tipo texto.
    [type="text"] {
    border: 1px solid black;
}

  • Seletores de pseudo-classe: selecionam elementos em um estado específico. Exemplo: :hover para selecionar elementos quando o mouse estiver sobre eles.
    :hover {
    color: red; 
}
